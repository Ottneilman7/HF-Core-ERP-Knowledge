# HF Core ERP Knowledge

Repositorio oficial de conocimiento del proyecto **HF Core ERP**.

Este repositorio contiene toda la documentación funcional, técnica y arquitectónica utilizada durante el desarrollo del sistema.

El objetivo es garantizar que cualquier desarrollador, colaborador o asistente de IA pueda comprender el estado del proyecto y continuar el desarrollo sin depender del historial de conversaciones.

---

## Propósito

HF Core ERP nació como el ERP interno de Honestly Foods CA.

Su visión es evolucionar hacia una plataforma SaaS diseñada para Startups, Emprendedores y PyMEs, ayudándoles a controlar y hacer crecer su negocio mediante información útil para la toma de decisiones.

Este repositorio documenta cómo se construye ese producto.

- Bloqueos (si existen).

PROJECT_INDEX.md

- Reglas de negocio.
- Dependencias.
- Estado del desarrollo.

---

# Paso 5 — Revisar las decisiones de arquitectura

Leer los ADR relacionados con el módulo que será modificado.

Nunca implementar cambios que contradigan una decisión arquitectónica previamente aprobada.

Repositorio:

HF-CORE-ERP

# Antes de comenzar
